# Smart Hire Hub

Smart Hire Hub is a full-stack recruitment platform built to simplify the hiring process. Recruiters can post and manage job openings, while candidates can explore opportunities and apply for jobs through an easy-to-use interface.

## Features

* User authentication
* Recruiter and Candidate roles
* Job posting and management
* Job search and applications
* Responsive design
* Secure REST APIs

## Tech Stack

* **Frontend:** React.js, JavaScript, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Tools:** Git, GitHub, Postman, VS Code

## Getting Started

```bash
git clone https://github.com/praveen-gouraj/Smart-Hire-Hub.git
cd Smart-Hire-Hub
npm install
npm start
```

## About

This project was built to learn and implement full-stack web development concepts such as authentication, role-based access, CRUD operations, and API integration while creating a practical recruitment platform.
